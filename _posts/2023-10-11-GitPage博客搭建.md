# 2. 本地调试

安装[Ruby](https://rubyinstaller.org/)

打开CMD，执行命令安装Jekyll

```mipsasm
gem install jekyll
```

设置镜像源

```
bundle config mirror.https://rubygems.org https://mirrors.tuna.tsinghua.edu.cn/rubygems
```

进入仓库文件夹，执行命令

```mipsasm
bundle install
```

启动Jekyll服务

```mipsasm
bundle exec jekyll serve
```
