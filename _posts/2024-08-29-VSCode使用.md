# 2. 格式化代码

1. 下载插件 Prettier
2. 勾选 format on save
3. 打开配置文件setting.json，添加下面内容

```json
// 配置JavaScript的格式化工具
"[javascript]": {

    "editor.defaultFormatter": "esbenp.prettier-vscode"

},
// 配置JSX的格式化工具
"[javascriptreact]": {

    "editor.defaultFormatter": "esbenp.prettier-vscode"

},
```
