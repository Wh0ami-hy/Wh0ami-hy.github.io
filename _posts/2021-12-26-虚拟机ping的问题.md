# nat模式ping不通

* 重置vmware的网络适配器

* 检查防火墙

```
Windows XP，Windows Server 2003：
Windows防火墙 --> 高级 --> ICMP --> 设置 --> 把“允许传入回显请求”前打上勾 --> 确定 --> 大功告成

Windows 7，Win 2008 R2，2012 R2：
Windows防火墙 --> 高级设置 --> 入站规则 --> 在列表里找到“文件和打印机共享(回显请求 - ICMPv4-In)” --> 右击选择启用规则，并在其属性里设置为‘允许连接'--> 确定 --> 大功告成

```
